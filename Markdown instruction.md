## Списки

Чтобы добавить ненумерованные списки, необходимы пункты выделить звёздочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добаавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
